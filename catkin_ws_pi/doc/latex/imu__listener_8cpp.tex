Function that gets called whenever new I\+MU Data is received.~\newline
I\+MU 1\+: Side I\+MU. Gathers x component of rotation speed.~\newline
I\+MU 2\+: Bottom I\+MU. Gathers z component of rotation speed.~\newline
I\+MU 3\+: Front I\+MU. Gathers y component of rotation speed.~\newline
The callback function combines the z components of the single I\+M\+Us to combine them into one virtual I\+MU that is placed in the middle of the sphere.
